# 04 统一接口设计

## 文档目标
- 定义上层无感的控制接口标准。

## 统一接口草案
- 状态：`/joint_states`（`sensor_msgs/JointState`）
- 轨迹：`FollowJointTrajectory`（`control_msgs`）
- 控制器：`controller_manager` + `JointTrajectoryController`

## 统一约束
- 关节命名一致（URDF / 驱动 / MoveIt 同名）
- 单位一致（位置 rad、速度 rad/s）
- 限位一致（软限位与控制器参数一致）
